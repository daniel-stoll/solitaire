# solitaire
A simple clone of solitaire inspired by the windows xp version.
I do really like solitaire (or Klondike) and I also really hate ads interupting my gameplay.
The game does not collect any data on the user nor does it send anything to a remote server.

For performance reasons, the local storage is used.

Just play as you wish in the difficulty that you prefer (or random). 
Have fun!

Disclaimer: this is 100% vibe coded, no single line has been touched by a human.
It can contain bugs. Open an issue in this case
